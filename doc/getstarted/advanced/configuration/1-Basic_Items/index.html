Basic Items of Configuration

The configuration file is basically a JSON file, while we allow comments inside it.

config fieldoptionalavailable valuesdescription
typenocenter / hubset to center if it is a cetner, or hub if it is a hub
nameyesgive a meaningful name for this center or hub. It's just a name easy-to-read so no need to be universally unique
idyesthis should be universally unique as it refers to this center or hub. If omitted, system would try to generate one for it
